# 隐私政策

生效日期：2026-07-12  
最后更新：2026-07-12

桌面便签（StickyNote，本地开源版）由 miczhang007（个人开发者）提供。本政策说明当前本地版如何处理数据。

## 数据处理

当前版本不收集、不上传、不出售或与第三方共享个人信息、使用行为数据或便签内容。应用不提供账号注册、广告、统计分析或云同步服务。

你创建的便签内容、截止时间、重要标记及相关操作记录仅保存在当前 Windows 设备的本地 SQLite 数据库中。应用不会将这些数据传输至开发者服务器或其他第三方服务。

## 本地存储与删除

应用数据通常保存在 Windows 当前用户的应用数据目录中，数据库文件名为 `desktop-sticky-note.sqlite3`。卸载应用未必会删除该数据文件；如需彻底删除数据，请先退出应用，再删除对应的应用数据目录或数据库文件。

## 第三方服务与权限

当前版本不使用广告、分析、崩溃监测、云存储或第三方登录服务。应用仅使用运行所需的本地文件存储能力；如你开启开机自启动，应用会在当前 Windows 用户的启动项中创建对应设置。

## 未来云服务

未来如推出云同步或云端备份服务，该服务将作为独立、可选的订阅服务提供。启用前会提供更新后的隐私政策，明确账户信息、数据类型、存储地点、保留期限及删除方式；在政策更新前，当前本地版的数据处理方式不变。
