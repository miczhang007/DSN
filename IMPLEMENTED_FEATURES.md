# 桌面便签 / StickyNote - 已实现功能需求文档

## 1. 产品定位

面向 Windows 桌面的轻量本地待办便签工具。用户可在桌面上维护多个用户的待办事项，记录截止时间、紧急标记与执行进度。

- 当前应用展示版本：`v1.0.6 - 2026-07-17 17:45`
- 数据默认保存在本机 SQLite 数据库，不上传服务器。
- 支持 Microsoft Store MSIX 与便携 EXE 分发。

## 2. 任务管理

### 2.1 待办列表

- 展示当前用户的未归档任务。
- 排序规则：紧急任务优先；随后按截止时间升序；无截止时间任务靠后。
- 任务卡展示任务标题、截止时间与紧急标识。
- 空列表时提示用户添加首个任务。

### 2.2 创建任务

- 支持填写任务标题，最大 80 字。
- 支持设置可选截止时间。
- 支持设置紧急标记。
- 标题为空时不可提交。
- 创建任务时自动写入生命周期记录。

### 2.3 编辑与完成

- 支持编辑任务标题、截止时间与紧急标记。
- 对任务内容、截止时间、紧急状态的每次实际变更均会写入生命周期记录。
- 支持一键完成任务。
- 完成后任务进入历史任务列表，并同时记录“完成”“归档”事件。
- 支持从历史列表撤销完成，任务恢复为待办状态。

## 3. 进度维护与生命周期

- 在任务详情中可新增进度记录，单条最多 240 字。
- 进度记录适用于当前进行中的任务；已归档任务不可新增进度。
- 每条进度记录显示在生命周期列表中，包含记录时间与内容。
- 支持删除单条历史进度记录，删除前显示应用内确认弹窗。
- 生命周期自动记录：
  - 创建任务
  - 标题变更
  - 截止时间变更
  - 紧急标记变更
  - 进度新增
  - 完成、归档与撤销完成
- 普通生命周期记录不可删除，仅进度记录支持删除。

## 4. 用户管理

- 支持创建多个本地用户，用户名最长 40 字。
- 支持切换当前用户；不同用户的任务与生命周期记录相互隔离。
- 支持重命名用户，并同步迁移该用户的所有任务数据。
- 支持删除用户；删除时会删除其待办、历史任务和生命周期记录。
- 危险操作使用应用内确认弹窗，不使用系统原生浏览器提示框。

## 5. 桌面与窗口设置

- 支持便签大小：小、中、大。
- 支持便签主题：黄色、蓝色、绿色、粉红、白色。
- 支持便签位置：居中、右上、右下、左上、左下。
- 支持开机自启动开关，使用当前 Windows 用户注册表启动项。
- 支持极简模式：
  - 隐藏系统窗口外框；
  - 隐藏任务栏入口；
  - 移除便签外部留白、边框和阴影。
- 非极简模式保留普通窗口外框及白色外沿。
- 主题、大小、位置、极简模式与当前用户选择均保存于本机。

## 6. 产品信息与隐私

- 产品简介展示产品名称、当前版本、开发者、数据说明、源码仓库、隐私政策与联系方式。
- 数据说明明确：便签数据默认保存在本机，不上传至服务器。
- 提供 GitHub 源码仓库和隐私政策链接。

## 7. 技术与数据

- 前端：Vue 3 + Vite。
- 桌面端：Tauri 2。
- 本地数据库：SQLite。
- 数据表：
  - `tasks`：任务主体、所属用户、时间、紧急状态、完成/归档状态。
  - `task_events`：任务生命周期与进度事件。
- 启动异常会写入 `%LOCALAPPDATA%\DesktopStickyNote\startup.log`，用于排查启动失败。

## 8. 当前不包含

- 云同步、账号登录、跨设备数据共享。
- 系统通知或截止时间提醒。
- 任务删除功能。
- 任务分类、标签、重复任务、附件。
- 多设备备份与恢复。
